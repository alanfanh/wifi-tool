# wifi-test-tool

> PyQt project.

## 介绍

基于PySide6、pywif开发的WiFi测试工具，用于验证无线客户端反复断开连接无线信号对路由器的影响。

### Developer

[FanHao](http://alanfanh.github)

## 环境

> 运行此源码的PC上需具备无线网卡

### 语言

```text
python3.10 64bit
```

### 依赖

> 可使用"pip install -r requirements.txt"一键安装所有依赖项

````text
pyside6==6.2.1
pywifi==1.1.12
configobj==5.0.6
comtypes==1.1.10
````
